# Sistema de Gestión de Clientes, Servicios y Reservas

## 📌 Descripción
Este proyecto implementa un sistema en Python basado en programación orientada a objetos para gestionar clientes, servicios y reservas sin el uso de bases de datos.

---

## 🎯 Objetivos
- Aplicar abstracción, herencia, polimorfismo y encapsulación
- Implementar manejo avanzado de excepciones
- Garantizar estabilidad del sistema ante errores

---

## 🧱 Estructura del sistema

### Clases:
- Cliente
- Servicio (abstracta)
- Reserva

### Tipos de servicios:
- ReservaSala
- AlquilerEquipo
- AsesoriaEspecializada

---

## ⚠️ Manejo de excepciones
Se implementaron:
- ErrorCliente
- ErrorServicio
- ErrorReserva

Uso de:
- try / except
- try / except / else
- try / except / finally

---

## 📁 Registro de logs
Los errores y eventos se almacenan en:
- logs.txt
